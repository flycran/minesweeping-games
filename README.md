# 扫雷小游戏

#### 介绍
web技术开发的前端扫雷小游戏。
#### 基本玩法

更改难度和尺寸会自动重启游戏

单击揭示该格

右击标记/取消该格为红色

开局拥有一定无敌次数，无论是否踩雷都会消耗剩余无敌次数

踩到雷结束游戏，自动重启游戏

揭示所有非雷区，游戏结束

#### 技术参数

最小尺寸：8*8

最大尺寸：36*36

最小难度：3

最大难度：30

无敌步数：向上取整（难度/6）
